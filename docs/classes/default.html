    Class default

    RBAC (Role-Based Access Control) class to manage permissions. This class provides a simple way to add permissions and check if a given role has permission to perform an action on a resource. It is intentionally minimalistic for example/POC use.

    Example

    const rbac = new RBAC();
    rbac.addPermission('admin', ['create','read'], ['resource1']);
    rbac.checkPermission('admin','create','resource1'); // true
